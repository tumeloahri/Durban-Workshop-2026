Basic boxplot — age by study arm

ggboxplot(table_01, x = "arm", y = "age")

Add pairwise comparison

ggboxplot(table_01, x = "arm", y = "age") +
  stat_compare_means()

The default test is Wilcoxon rank-sum test (Mann–Whitney U) for 2 groups.

With points + color

ggboxplot(table_01, x = "arm", y = "age",
          add = "jitter",
          color = "arm", palette = c("#E7B800", "#00AFBB")) +
  stat_compare_means()

Facet by education

Now let’s see the same plot split by education level using facet_wrap().

ggboxplot(table_01, x = "arm", y = "age",
          add = "jitter",
          color = "arm", palette = c("#E7B800", "#00AFBB")) +
  stat_compare_means() +
  facet_wrap(~ education)

Here facet_wrap(~ education) creates one small panel for each education level, so you can see whether the age pattern differs across education groups.

Two-way faceting: education × birth control

You can also facet by two variables with facet_grid():

ggboxplot(table_01, x = "arm", y = "age",
          add = "jitter",
          color = "arm", palette = c("#E7B800", "#00AFBB")) +
  stat_compare_means() +
  facet_grid(birth_control ~ education)

facet_grid(birth_control ~ education) makes a grid where rows are birth control types and columns are education levels.
