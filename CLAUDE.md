# Workshop Workspace — Intro to R for Data Science with LLMs

This is my personal workspace for the **Intro to R for Data Science with
LLMs** workshop. Claude Code loads this file automatically so it knows how to
help me.

## About me

I am a **beginner**. Please explain things simply, avoid jargon, and when you
do use a technical term, briefly say what it means.

## How this project is organized

- `notebooks/` — starter notebooks for the workshop. These are Quarto `.qmd`
  files: a mix of plain-language text and runnable R code, rendered to HTML
  with Quarto.
- `data/` — put any data files I work with here.

## Working with me

- When suggesting code, keep it short and explain what each part does.
- If something can be done more than one way, pick the simplest one.
- Feel free to suggest small experiments I can try to learn.
